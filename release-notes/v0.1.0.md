# v0.1.0

First public CLI-first release of WPS Meeting Archive.

## Highlights

- Python CLI for syncing recent meetings into `会议待归档表`
- `finalize-confirmed` flow for writing approved items into `文音视频档案`
- Automatic refresh of `user access_token` using `refresh_token`
- Windows `.bat` and macOS `.command` launchers for double-click usage
- Cross-platform README for macOS and Windows 11

## Notes

- Requires Python 3.9+
- First authorization is manual
- Later runs automatically refresh the user token until `refresh_token` expires
- The tool still cannot read manually renamed “会议记录” titles through public WPS APIs
